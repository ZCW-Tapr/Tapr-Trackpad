Reference guide for event codes from the trackpad that runs from the code:

Code 57 (ABS_MT_TRACKING_ID): Finger tracking. Positive = finger down, -1 = finger lifted
Code 53 (ABS_MT_POSITION_X): Finger X position
Code 54 (ABS_MT_POSITION_Y): Finger Y position
Code 47 (ABS_MT_SLOT): Which finger (0 = first, 1 = second, 2 = third)
Code 330 (BTN_TOUCH): 1 = touching, 0 = released
Code 325 (BTN_TOOL_FINGER): 1 finger on pad
Code 333 (BTN_TOOL_DOUBLETAP): 2 fingers on pad
Type 0, Code 0 (SYN_REPORT): End of event batch (ignore these)
Type 4, Code 5 (MSC_TIMESTAMP): Timestamps (ignore these)
